# Sistema TO DO – Organizador de Tarefas <img src="https://logo.svgcdn.com/devicon/python-original.svg" alt="Python logo" width="24" /> 
## Objetivo

O objetivo deste projeto é implementar um organizador de tarefas funcional, totalmente operado pelo terminal, utilizando estruturas de dados como listas ou dicionários, podendo também salvar tarefas em arquivos.
O sistema oferece comandos claros e funções específicas para cada operação, garantindo organização e facilidade de manutenção.

## Estrutura e Armazenamento

As tarefas podem ser armazenadas utilizando:

- Listas

- Dicionários

- Arquivos de texto (opcional)

Cada tarefa possui:

- ID único

- Descrição

- Prioridade

- Status (pendente ou concluída)

## Arquitetura do Código

O sistema utiliza funções específicas para cada operação, como:

- adicionar_tarefa()

- listar_tarefas()

- editar_tarefa()

- concluir_tarefa()

- remover_tarefa()

- salvar_dados()

- carregar_dados()

Essa organização facilita a leitura, manutenção e expansão do código.

### Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/gabrielvinte/Sistema-TO-DO-Organizador-de-Tarefas.git
```
2. Acesse o diretório do projeto:
```bash
cd Sitema-TO-DO-Organizador-de-Tarefas
```
3. Execute o sistema:
```bash
python main.py
```
